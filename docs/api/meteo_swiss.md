# meteo_swiss

Find the nearest MeteoSwiss automatic weather station for a given location and sensor type.

Fetches live station data from geo.admin.ch and selects the closest station that has the requested sensor (temperature, global radiation, relative humidity, or rain) within a 150 m altitude tolerance.
