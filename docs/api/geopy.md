# geopy

Geocoding and coordinate utilities for Swiss and global use cases.

Includes WGS84-to-LV95 conversion, altitude lookups (opentopodata.org and geo.admin.ch), address geocoding via Nominatim, Swiss postal code resolution via pgeocode, and Haversine distance calculation.
