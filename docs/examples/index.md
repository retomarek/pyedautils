# Examples

Usage examples for all pyedautils modules. See {doc}`../api/index` for the full API reference.

- {doc}`data_io` — Save and load data in CSV, pickle, JSON
- {doc}`geopy` — Geocoding, coordinate conversion, altitude lookup
- {doc}`plots` — Visualizations (Daily Profiles, Mollier h,x-diagram)
- {doc}`data_prep` — Data preprocessing (season detection)
- {doc}`weather` — Swiss weather data (MeteoSwiss, Agrometeo.ch)
